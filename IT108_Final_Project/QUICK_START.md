# Quick Start Guide

## For Administrators

### First Time Setup
1. Start XAMPP (Apache + MySQL)
2. Import `database/schema.sql` in phpMyAdmin
3. Access: http://localhost/IT108_system/
4. Login: admin / admin123
5. **Change admin password immediately!**

### Daily Operations
- **Monitor Catches**: View dashboard for daily statistics
- **Verify Records**: Review pending catch records
- **Manage Users**: Add/edit fishers and officers
- **Generate Reports**: Export data for analysis

## For Fishers

### Getting Started
1. Register an account on the login page
2. Complete your profile information
3. Register your fishing vessels
4. Start recording catches

### Recording a Catch
1. Click "Record Catch" in navigation
2. Fill in the catch details:
   - Date and time
   - Species caught
   - Quantity and unit
   - Price (optional)
   - Vessel used
   - Fishing gear
   - Location/zone
3. Click "Record Catch"
4. Wait for officer verification

### Managing Vessels
1. Go to "My Vessels"
2. Click "Add Vessel"
3. Enter vessel details
4. Save

### Viewing Records
- Go to "Catch Records" to see all your catches
- View statistics on Dashboard
- Check monthly trends

## For Officers

### Verifying Catches
1. View all catch records
2. Review pending records
3. Verify or reject catches
4. Add notes if needed

### Monitoring

## Tips

1. **Always record catches immediately** after fishing
2. **Be accurate** with quantities and species
3. **Include location** for better tracking
4. **Check vessel status** before recording
5. **Review your records** regularly
